薛定谔的Claude

本页全文处于叠加态——在你观测之前，每个字都是随机的乱码。移动光标扫过文字，波函数坍缩，真实内容随之显形。

机制 · Mechanic

角色选择（菜单栏「勇士」）里藏着一张人影卡片——薛定谔的 Claude。它处于叠加态：在你观测之前，谁也说不清里面是哪一位。点击人影即坍缩，随机显形成一位 Claude 勇士；再点一次，就重新掷一次。选中的 Claude 会像其它勇士一样出现在副本战斗中，并自动存档。

名册 · 六重叠加

  • ✳贤者 Claude—赭橙长袍，掌心悬着一枚星芒
  • ✳骑士 Claude—珊瑚橙铠甲，盾上镌着星芒纹章
  • ✳法师 Claude—陶土法袍，指尖绽开橙色星芒
  • ✳游侠 Claude—赭色皮甲，箭尖凝着琥珀星火
  • ✳武僧 Claude—米白缠布，拳间萦绕星芒真气
  • ✳薛定谔之猫 Claude—橘猫法师，半透明的爪子藏着叠加态

观测装置 · Generate

$npm run gen:claude
$npm run opt:claude
$python3 scripts/cutout-assets.py --dir claude

增量 · 非破坏

立绘存放在专用文件夹 src/assets/claude/，与装备图标一样用 OpenRouter 生成，提示词见 scripts/asset-manifest.mjs 的 CLAUDE 清单。没生成立绘时，人影仍可玩——卡面回退到 emoji 字形，战斗中回退到默认勇士立绘（见 src/assets/claudes.ts，通过 import.meta.glob 自动匹配）。

观测者效应：你读到的每一个字，都是为你而坍缩的。移开光标，它便重返叠加态。
